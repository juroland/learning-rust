# Learning Rust

This repository tracks my progress in learning Rust.

## Current Progress

### Fundamental Concepts Learned

- **Generic Functions** - Type parameters with trait bounds (`Ord`, `Copy`, `ToPrimitive`)
- **Trait Bounds** - Constraining generic types to support required operations
- **Iterator Trait** - Implementing custom iterators for idiomatic Rust
- **Option & Result Types** - Handling optional values and errors
- **Error Handling** - Using `?` operator for error propagation
- **Ownership & Borrowing** - References, moves, and the `Copy` trait
- **Iterators** - `.iter()`, `.map()`, `.sum()`, `.collect()` for functional operations
- **Type Casting** - Safe numeric conversions with `ToPrimitive` trait
- **I/O Operations** - Buffered reading with `BufReader`
